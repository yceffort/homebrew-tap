# yceffort/tap

Homebrew tap for [yceffort](https://github.com/yceffort) projects.

## Install

```bash
brew tap yceffort/tap
```

## Available casks

- `yc-clipboard` — [yc.clipboard](https://github.com/yceffort/mac-clipboard), a lightweight macOS clipboard manager.

## Usage

```bash
brew install --cask yc-clipboard
brew upgrade --cask yc-clipboard
brew uninstall --cask yc-clipboard
```
